# 智能泡茶机 M1 错误码 E04：网络连接异常

> **模拟资料说明**：本错误码说明用于测试智能设备、APP 与售后检索链路。真实错误码定义及处理要求请以正式产品说明和客服系统为准。

## E04 的含义、排查步骤与升级条件

**E04：网络连接异常。** 首次出现时，先停止重复点击或反复启动，记录 APP 或设备屏幕上的完整提示。建议按“安全检查→基础复位→部件确认→再次尝试”的顺序处理。检查设备是否离线、Wi-Fi 是否为可用的 2.4GHz 网络、APP 权限是否开启。

1. 先确认设备周边干燥、平稳，电源和水位等基础条件满足；带电设备不要在漏水、异味、异常发热或外壳破损时继续操作。
2. 关闭当前任务，等待设备停止后再检查可拆卸部件。水箱、茶仓、滤网、杯盖、底座、网络或账号状态都应与当前错误提示对应；不要一次同时拆卸多个部件，以免无法判断问题来源。
3. 完成检查后，可断电或退出 APP 约 10 秒，再重新开启并按正常流程尝试一次。若提示消失且设备能完成一次完整任务，可继续观察；若同一错误在短时间内反复出现，应停止连续重试。
4. 需要售后时，提交订单号、设备型号、E04 截图/照片、出现时间、已尝试的操作和异常现象。若涉及冒烟、焦味、明显漏水、插座发烫或人身安全风险，应立即断电并联系人工售后。

**不要这样做**：不要自行拆机，不要使用非原配或损坏电源，不要在升级、加热或清洗过程中强制拔电源，不要把错误码当作唯一故障结论。E04 仅用于初步定位，最终处理以设备检测和售后审核为准。

如果同一提示在完成基础检查后仍出现，请不要通过反复开关机、连续多次提交、强制拔电源或拆卸内部部件来“碰运气”。更有效的做法是记录该错误首次出现的条件，例如是否刚加水、刚更换网络、刚升级 APP、刚支付或刚安装滤网，并记录设备当前指示灯、页面提示和已尝试的步骤。这些信息能帮助售后判断是环境问题、部件状态、账户状态还是设备本身需要进一步检测。

错误码答复应保持边界：它用于指导用户完成低风险自查，不等同于远程诊断或最终责任认定。若涉及订单金额、退款、保修期限、物流状态或会员权益，应分别检索对应政策；若问题涉及账户安全，应先保护账号并通过官方入口处理。无论是设备还是 APP，只要出现安全风险、重复失败或用户无法完成基础功能，都应转人工，而不是无限循环同一套步骤。
